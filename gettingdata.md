# “Getting” Data: Questioning, Creating, Sourcing 

Before jumping into a new project there are questions to grapple with on the large scale about the work you plan to do and the tools you plan to use (all of which will still be considered more deeply as a project progresses). Data are complex and come with baggage, as do the visualization methods and the tools we choose to use. 

This section offers a few basic questions as a jumping off point for starting a project, as well as a few preliminary considerations on data acquisition: sourcing and creating data.
